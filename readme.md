# RECON

HTTP API service for playing with machine learning models

## Launch service

```shell
pip3 install -r requirements.txt
python3 ./app.py
```

## Endpoints


| PATH       | Desc  |
|------------|:------|
| /          | infos |
| /res50     | ResNet50 pretrained model |
| /ageGender | Age and Gender estimator  |
